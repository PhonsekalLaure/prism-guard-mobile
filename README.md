# Prism Guard Mobile

Prism Guard Mobile is an Expo development-client app. Use a development build for normal testing, especially for push notifications. Expo Go is not enough for Android remote push notifications in this project.

## Setup

Read the full setup and run guide:

```txt
MOBILE_SETUP.md
```

Quick local start after dependencies and a development build are already installed:

```powershell
npm install
npx expo start --dev-client
```

Open the installed Prism Guard development app on a physical phone or Android emulator.
